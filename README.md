Finance Dashboard UI

A clean and interactive finance dashboard built using React.
This project helps users track their financial activity, analyze spending patterns, and manage transactions efficiently.

 Features
Dashboard Overview

* Displays **Total Balance, Income, and Expenses**
* Includes:

  * 📈 Time-based chart (balance trend)
  * 🥧 Category-based chart (spending breakdown)

 Transactions Section

* View all transactions with:

  * Date
  * Amount
  * Category
  * Type (Income/Expense)

* Functionalities:

  * 🔍 Search by category
  * 🔽 Filter (Income / Expense)
  * ➕ Add Transaction (Admin only)
Role-Based UI

* **Viewer**

  * Can only view data

* **Admin**

  * Can add transactions
  * UI changes dynamically based on role

Insights Section

* Smart analytics including:

  * 🔥 Highest spending category
  * 💰 Total savings
  * 📅 Monthly expense
  * 📉 Expense ratio
  * 💸 Average expense
  * ⚠️ Overspending alert

UI & UX

* Clean and modern design
* Fully responsive (Mobile, Tablet, Desktop)
* Light mode & Dark mode 🌙☀️
* Interactive cards and charts


 Approach

* Built using **component-based architecture**
* Separated pages:

  * Dashboard
  * Transactions
  * Insights
* Used **React state management (useState)** for:

  * Transactions data
  * Filters
  * Role handling
* Used **mock data** for simplicity (no backend)
* Focused on:

  * Clean UI
  * Usability
  * Real-time updates

Tech Stack

* React.js
* CSS (Custom styling)
* Recharts (for charts)




Conclusion

This project demonstrates frontend development skills including UI design, state management, responsiveness, and user experience.
